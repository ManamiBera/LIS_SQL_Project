# Library Data Analysis using SQL
## Overview 
In this project, I analyzed a library dataset using SQL to understand book usage, user activity and demand patterns.

## Analysis & Key Insights

### Basic Analysis

#### Questions
1. What is the total number of books in the library?

2. What is the total number of copies?

3. How many books have been issued?

#### Key Insights:

• The library contains 20 books and 60 total copies.

• A total of 21 books have been issued.

### User Activity Analysis

#### Questions:

1. Which members borrowed the most books?

2. Are there inactive members?

#### Key Insights

• A small number of members account for most borrowings.

• Many members have not issued any books.
